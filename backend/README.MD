# JudgeX

JudgeX is a modern online coding platform inspired by LeetCode. It enables users to solve programming challenges, execute code securely, track their progress, and improve problem-solving skills.

## Features

* JWT Authentication
* Email Verification
* Password Reset via Email
* Role-Based Access Control (RBAC)
* Admin Management
* Secure Password Hashing with bcrypt
* MongoDB Database
* Express.js REST APIs
* Judge0 Code Execution Integration
* Scalable Backend Architecture

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* Nodemailer
* bcryptjs
* Judge0 API
